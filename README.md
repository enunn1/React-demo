# Freelance Portfolio & Lead Intake Platform

A modern portfolio website built with Next.js and React to showcase featured projects and support freelance development inquiries.

Designed to highlight full-stack experience, present curated work, and provide a clean, responsive contact experience.

---

## Features

- Curated Featured Projects section
- Dynamic GitHub repository integration
- Dedicated project detail pages
- Responsive design for desktop and mobile
- Dark / Light theme support
- Contact form with server-side email delivery

## Tech Stack


- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Resend (email delivery)
- GitHub REST API

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
